Password Strength Checker Using Python

In this project, the user creates a password and the program checks the strength of the password based on certain criteria. The password is evaluated based on the following criterias:

  • Length: How long the password is
  • Letters: The amount of uppercase and lowercase letters used
  • Digits: The presence of numeric characters
  • Special Characters: The presence of special symbols like & or $
The program assigns a score to the password based on those criterias and categorizes it as "Weak", "Good", or "Strong". This helps users understand the importance of creating strong passwords and encourages them to do so for their own online security.
